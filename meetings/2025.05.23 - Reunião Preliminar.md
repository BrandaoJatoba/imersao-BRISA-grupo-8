# ATA DE REUNIÃO PRELIMINAR
Data: 23 de maio de 2025
Projeto: Gerenciamento de Combustível
Cliente: SENAI
Participantes: Equipes do SENAI e do Projeto BRISA
Tipo de Reunião: Reunião preliminar para levantamento de informações e definição de direcionamentos iniciais.

## 1. Propósito da Reunião
A reunião teve como objetivo alinhar expectativas e levantar requisitos iniciais para a evolução do sistema de gerenciamento de frota utilizado pelo cliente SENAI, no contexto do Projeto BRISA.

## 2. Situação Atual
O SENAI já possui um sistema de gerenciamento de frota em operação.

Há um controle de combustível implementado por meio de registros de abastecimento.

Foi identificada a necessidade de um controle mais específico e detalhado das operações da frota.

## 3. Funcionalidades Extras Sugeridas (não previstas inicialmente)
Durante a reunião, foram discutidas possíveis novas funcionalidades a serem incorporadas ao sistema, tais como:

Chat interno entre usuários da ferramenta.

Integração com GPS, permitindo o rastreamento em tempo real da localização dos motoristas.

Compartilhamento de Viagem (carona), visando otimização de recursos e sustentabilidade.

## 4. Considerações Técnicas e Estratégicas
A proposta é implementar as melhorias sobre o sistema existente, mantendo sua base.

A frota atual é composta por aproximadamente 50 veículos, entre carros e vans.

A equipe do Projeto BRISA ficou responsável por sugerir o hardware necessário para suporte às novas funcionalidades.

O sistema atual é disponibilizado via Web e PWA (Progressive Web App).

O projeto tem potencial de replicação para outras unidades do SENAI além da unidade atual.

Aguardar próxima reunião in loco para mais detalhes sobre o sistema existente.
